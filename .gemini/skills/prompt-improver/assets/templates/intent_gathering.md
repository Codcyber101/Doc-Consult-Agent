# Intent Gathering Questionnaire

Use this template when the user's request is too vague (e.g., "Make a website" or "Fix code").

**Introduction:**
"To generate the best prompt for you, I need to clarify a few details. Please answer the following (or press Enter to skip):"

## 1. The Core Goal
- **What are we building/fixing?** (e.g., A landing page, a sorting algorithm, a database migration).
- **Input/Starting Point:** Do you have existing code, or is this from scratch?

## 2. The Tech Stack
- **Language/Framework:** (e.g., Python/Django, React/Next.js).
- **Libraries:** (e.g., Tailwind CSS, Pandas, SQLAlchemy).

## 3. The Desired Output (Variant Selection)
Which approach do you prefer?
1.  **Fast & Direct:** Just give me the code.
2.  **Robust & Production-Ready:** thorough error handling, tests, and best practices.
3.  **Educational:** Explain how it works so I can learn.

## 4. Constraints
- **Platform:** (e.g., Web, Mobile, CLI).
- **Restrictions:** (e.g., No external libraries, Must run on Windows).

---
**Example Usage by Agent:**
"I see you want to build a 'Task App'.
1. **Stack:** Is this React, Python, or Mobile?
2. **Style:** Do you want a simple prototype (Option 1) or a scalable app with database integration (Option 2)?"
