# Obsidian Templating Guide

## Core Features
- **Placeholders**: Use {{date}}, {{time}}, etc. for dynamic content.
- **Frontmatter**: YAML for metadata like tags, type, status.
- **Sections**: Organize with headers for readability.
- **Links**: Internal [[Note]] or external [Link](url).

## Advanced Features
- **Templater Plugin**: Conditionals <% if (...) %> and JavaScript execution.
- **Dataview Plugin**: Queries like `LIST FROM "Folder" WHERE condition`.
- **Kanban Plugin**: Boards for task management.
- **Calendar Plugin**: Date-based automation.

## Best Practices
- Use consistent structures.
- Link notes for integration.
- Customize with plugins for power users.
