# Placeholder Cheatsheet

## Core Placeholders
- {{date}}: Current date (YYYY-MM-DD)
- {{time}}: Current time
- {{title}}: Note title
- {{yesterday}}: Yesterday's date
- {{tomorrow}}: Tomorrow's date
- {{week}}: Week number
- {{month}}: Month name
- {{year}}: Year

## Advanced (with Plugins)
- Templater: <% tp.date.now() %>
- Dataview: Inline queries
- Custom: Define in Templater settings

## Usage
Replace in templates for auto-fill.
