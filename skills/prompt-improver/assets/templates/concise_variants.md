# Concise Prompt Variants Template

Use this template to generate simplified, high-impact prompt options for the user.

**Input:**
- **Intent:** [User's Goal, e.g., "Create a regex for email"]
- **Stack:** [Tech Stack, e.g., "JavaScript"]

**Output:**

Here are 3 concise prompt variants for your request. Copy the one that fits your need:

### Option A: The Executor (Fast & Direct)
```text
Act as a code generator.
Task: Create a [Stack] regex for [Intent].
Constraints: Provide the regex pattern and a usage example only. No explanation.
```

### Option B: The Architect (Robust & Secure)
```text
Act as a Senior [Stack] Engineer.
Task: Implement a robust [Intent].
Constraints: Handle edge cases (e.g., unicode, subdomains), ensure security/performance, and include unit tests.
```

### Option C: The Mentor (Educational)
```text
Act as a [Stack] Mentor.
Topic: [Intent].
Task: Explain how [Intent] works step-by-step. Break down the syntax/logic. Then provide a commented example.
```
